# College of Dentistry Revise

An interactive dentistry web app for exploring dental anatomy with a polished
3D-focused interface. This project is built as a lightweight static site and is
served locally with a small Node.js server.

## Highlights

- 3D dental anatomy viewing experience
- Responsive desktop and mobile-friendly layout
- Static frontend that is simple to run locally
- Included local Node.js server for quick preview and testing

## Project Structure

- `index.html` - main app shell
- `main.css` - primary styling
- `mobile.css` - mobile-specific styling
- `mobile.js` - mobile interaction logic
- `dentistry_js.js` - main application behavior
- `serve.js` - local static server

## Run Locally

Requirements:

- Node.js 12 or newer

Start the app:

```bash
cd "/Users/bobbydhir/Desktop/Dental anatomy/Dental"
npm install --no-audit --no-fund
npm start
```
